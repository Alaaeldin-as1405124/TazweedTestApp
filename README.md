# TazweedTestApp
  React Native Booking Application (Localized (Ar, en))
  
 Application functions
  - Register buyer account
  - Login
  - View sellers list
  - Search in seller list
  - Book Appointment
  - View My Appointments
  - Filter My Appointments
  - Ability to switch the language
